## Como Instalar

### Instalación

`npm install`

ó

`yarn install`

### Correr Backend fake para ver datos mock

`npm run fake-api`

ó

`yarn run fake-api`

### Correr Proyecto Frontend

`npm run start`

ó

`yarn run start`

### Debe tener un JWT válido para iniciar sesión

`La aplicacion le enviará a la sesión de inicio de sesión al momento de que expire el actual`
